## 🚀 Cómo Empezar

### Paso 1: Configuración (5 min)
```bash
# Cambia IP en src/lib/axios.ts
# Ver: CONFIGURACION_RAPIDA.md
```

### Paso 2: Inicia Backend (Terminal 1)
```bash
cd personal-finance-api
npm run start:dev
```

### Paso 3: Inicia App (Terminal 2)
```bash
cd finance-mobile
npm start
# Presiona 'a' para Android
```

### Paso 4: ¡Listo! 🎉
```
App debe cargar sin errores
- Dashboard muestra números reales
- Transacciones carga desde API
- Categorías muestra lista
```

## 💡 Tips Finales

🔹 **Hot Reload**: Los cambios se aplican automáticamente (Fast Refresh)
🔹 **Debugging**: Presiona `d` en la terminal para DevTools
🔹 **Logs**: Los errores aparecen en rojo en la terminal
🔹 **IP Local**: IMPORTANTE cambiar en axios.ts
🔹 **Backend**: Debe estar corriendo en npm run start:dev
🔹 **WiFi**: Teléfono real debe estar en misma WiFi
